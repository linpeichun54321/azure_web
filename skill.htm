我的個人技能樹

1. SQL Server Analysis Services (SSAS)

擅長建置多維度資料模型（Multidimensional）與表格式模型（Tabular），優化大規模數據的查詢效能。

2. SQL Server Reporting Services (SSRS)

具備設計與管理企業級分頁報表的能力，能串接多種資料源並實現自動化報表排程發送。

3. Power BI

能從數據清洗 (Power Query)、建模到前端視覺化看板設計，提供具備洞察力的互動式報表。

4. DAX 開發

精通 Data Analysis Expressions，能撰寫複雜的計算資料行與量值 (Measures)，處理時間智能分析與進階篩選邏輯。

5. Python 開發

運用 Python 進行資料爬蟲、自動化腳本撰寫以及使用 Pandas/NumPy 進行高效數據清理與分析。

6. Azure SQL Database 建置

具備雲端資料庫架構設計經驗，包括資料庫建立、安全性管理（Firewall/VNet）以及效能調教。
